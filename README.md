# SQL Data Management Project

## Overview
This project demonstrates SQL skills through a Hospital Management Database. It contains 30 real-world SQL queries covering data retrieval, filtering, joins, aggregations, grouping, subqueries, and analytical insights.

## Skills Demonstrated
- SELECT Statements
- Filtering with WHERE
- ORDER BY
- INNER JOIN
- LEFT JOIN
- GROUP BY
- HAVING
- Aggregate Functions
- Subqueries
- Data Analysis

## Database Tables
- Patients
- Doctors
- Admissions
- Province Names

## Key Insights
- Patient demographics analysis
- Doctor performance tracking
- Admission trends
- Average hospital stay calculations
- Province-wise patient distribution
- Most common diagnoses analysis

## Technologies Used
- SQL
- SQLite
- VS Code
